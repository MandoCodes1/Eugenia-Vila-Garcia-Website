## Deployment

This is a static website that can be deployed to:
- **Cloudflare Pages** (recommended)
- GitHub Pages
- Netlify
- Vercel
- Any static hosting service

Simply upload the entire `_site/` directory contents to your hosting provider.

## Updates

### To Update Reviews
Edit `data/reviews.csv` with new patient testimonials, then the data is embedded in the HTML.

### To Update CV
Replace `documents/cv.pdf` with the updated version.

### To Update Profile Photo
Replace `assets/images/profile.png` with a new professional photo (recommended: 800x800px, square format).
